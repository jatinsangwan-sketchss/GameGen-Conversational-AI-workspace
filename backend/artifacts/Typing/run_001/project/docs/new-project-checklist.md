# New Project Checklist

## 1. Duplicate template project
- copy starter project folder
- rename project folder
- rename project inside Godot if needed

## 2. Update docs
- fill `prd.md`
- fill `gdd.md`
- fill `ui-spec.md`
- update `implementation-brief.md`

## 3. Confirm milestone
- define current target milestone
- define current playable scope
- list what is out of scope

## 4. Review structure
- confirm scene folders
- confirm script ownership
- confirm systems needed for this game

## 5. Prepare implementation tasks
- define small coding tasks
- define allowed files
- define acceptance criteria

## 6. Start M1
- implement first playable core loop
- test manually
- log issues
