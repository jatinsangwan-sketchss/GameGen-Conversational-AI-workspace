# Milestones

## M0 - Starter Project Foundation
Create reusable project template, conventions, docs, starter scenes, and workflow rules.

## M1 - Core Gameplay Loop
Implement one simple playable mechanic with one fail condition and one score path.

## M2 - HUD / Fail / Retry
Add gameplay HUD, fail state UI, restart flow, and basic loop completion.

## M3 - Progression / Meta
Add a light progression layer, upgrades, or long-term reward loop.

## M4 - Content Scaling
Add more levels, variations, enemies, upgrades, missions, or content data.

## M5 - Polish / Testing
Improve feel, clarity, balancing, and iterate using playtest feedback.
